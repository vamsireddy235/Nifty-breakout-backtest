üöÄ Features

- Backtest intraday 30-minute breakout strategy
- Run for **single index** or **all major indexes**
- Visualize:
  - Weekday performance
  - Cumulative gains
  - Winning/Losing day ratios
  - Calendar heatmaps
- Compare strategy vs NIFTY 50 index
- Beautiful Streamlit UI with tables and charts
